# 微信公众号富文本复制工具

一个简单的网页工具，将HTML代码**自动清洗**并复制为富文本格式，可直接粘贴到微信公众号后台。

## 在线使用

**https://mingweichen.github.io/wechat-html-copier/**

## 功能

- 🧹 **自动清洗HTML** — 解决安卓APP空格空行问题
- 📋 将HTML代码复制为富文本（保留样式）
- 👁️ 实时预览效果
- 📱 支持移动端

## 为什么需要清洗？

微信公众号安卓APP编辑器和网页版内核不同：

| 平台 | 内核 | 空白处理 |
|------|------|----------|
| **网页版** | Chrome/Blink | 宽松，空白符合并 |
| **安卓APP** | X5（旧版Chrome） | 严格，空白符拆分为显式空行 |

### 清洗规则

1. 删除HTML注释 `<!-- -->`
2. 合并多余换行 `\r\n\n` → `\n`
3. 删除标签间空白 `> <` → `><`
4. 块级标签转 `<p>` — `div/h1-h6` → `<p>`
5. 删除空白 `<span></span>` 和 `<p></p>`
6. 合并连续 `&nbsp;`（超过2个合并为1个）
7. 合并连续 `<br>`（超过2个合并为2个）
8. 清理 `<p>` 首尾的 `<br>`
9. 清理 style 属性内的换行

## 使用方法

1. 将生成的微信公众号HTML代码粘贴到文本框
2. 确保「自动清洗HTML」选项已勾选
3. 点击「复制富文本」按钮
4. 前往微信公众号后台编辑器
5. 直接 Ctrl+V / Cmd+V 粘贴
6. 安卓/网页版显示一致，无多余空行

## 技术原理

- 使用 `navigator.clipboard.write()` API 将HTML内容以 `text/html` MIME类型写入剪贴板
- 复制前自动执行清洗规则，确保双端兼容
